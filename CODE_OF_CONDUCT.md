# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**
as its Code of Conduct.

The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

Concerns about conduct in this project's issues, pull requests,
or discussions can be reported to
**`info@datadrivenconstruction.io`**. Reports are handled
confidentially.

## Enforcement

Community-impact guidelines and the tiered enforcement process
are those published with Contributor Covenant 2.1.

## Scope

This Code of Conduct applies within all project spaces
(GitHub issues, pull requests, discussions, commit messages)
and when an individual is representing the project in public
spaces. Representation of the project may be further defined
and clarified by the project owner.
